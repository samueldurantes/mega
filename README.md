# susi

A simple viewer of the cutoff scores for all Brazilian higher education institutions participating in [SISU](https://acessounico.mec.gov.br/).

## Getting started

1. Install packages

```bash
$ yarn install
```

2. Run in the development environment

```bash
$ yarn dev
```
